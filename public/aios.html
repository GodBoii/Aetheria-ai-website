Know Me

A quick guide to what Aetheria can do for you.

  • Native Assistant: Launch Aetheria with the power button.
    Setup tip

    Settings → Apps → Default apps → Digital assistant → Aetheria. Set Power button action to Assistant.

  • Circle to Search: Find anything on screen fast.
    How to use

    Open Native Assistant → press & hold the Aetheria logo → circle anything to search.

  • Mindspace Tile: Save screen insights instantly.
    How to add

    Open Quick Settings → Edit tiles → add Mindspace → tap the tile to capture.

  • Tasks: Capture and track follow-ups fast.
    How to create

    Type your task → tap the Tasks button → save. Use “Add reminder” to set a time.

  • File Attachments: Add up to 20 files per message for richer context.
  • Thinking Mode: Deeper reasoning when you need step-by-step clarity.
  • Tools: Use built-in tools for search, writing, analysis, and workflows.
  • Sub-Agents: Specialist helpers for coding, research, design, and planning.
